(论文阅读笔记) A Survey on Complex Knowledge Base Question Answering Methods, Challenges and Solutions

发表于2021-08-20|更新于2021-08-21|论文阅读
|阅读量:

论文地址：https://arxiv.org/pdf/2105.11644.pdf

摘要

本论文主要阐述KBQA（知识库问答）的任务背景，然后介绍了解决复杂KBQA问题的两种主流方法，分别是基于语义解析（SP，semantic parse）和信息检索（IR，imformation retrieval）的方法。

然后本文回顾了这两类方法对典型挑战的解决方案，并总结了未来研究的方向。

引言

知识库（KB，Knowledge Base）是一个 *(subject, relation, object)*形式的的集合。知识库问答（KBQA，Knowledge Base Question Answer）是将知识库作为其知识源来回答自然语言问题。早期KBQA只在简单问题上进行应用，近期研究人员开始在KB上对复杂问题进行研究。

简单问题：即仅涉及单个（subject，relation，object）。

复杂问题：问题中包含多个subject从而涉及多跳推理，表达了复合关系以及数值运算。

基于语义解析（SP-based）的方法将问题表示成逻辑形式，然后在知识库中执行该逻辑形式从而得到答案；基于信息检索（IR-based）的方法构建一个问题相关地特定子图，从而获得与问题相关的综合信息，并根据这些信息对实体与问题之间的相关性进行排名，并选取排名高的实体。

将这两种主流方法应用到复杂问题的知识库问答上，在相应方法的不同部分带来了相应的挑战：

  • 在SP-based方法中使用的解析器难以涵盖各种复杂的查询（多跳推理、约束关系和数值运算等），在IR-based的类似方法可能无法回答复杂的查询，因为它们的排名是在没有可追溯推理的情况下在小范围实体上执行的。
  • 复杂的问题涉及的主题以及关系也更多，从而导致解析时潜在的逻辑形式存在着更大的搜索空间，这将极大增加计算资源的消耗。同时更多的主题和关系可能阻碍IR-based方法查找所有相关实体进行排名。
  • 两种方法首先都需要对查询问题进行理解，当问题在语法和语义上变得复杂起来，模型需要具有强大的自然语言理解和泛化能力。
  • 标记复杂问题答案的推理路径代价很大。通常仅仅只提供问答对，这表明SP-based和IR-based方法必须分别在没有正确逻辑形式和推理路径注释的情况下进行训练。这种弱监督信号对这两种方法也带来了相应的难题。

两种主流方法

SP-based方法遵循解析然后执行（parse-then-execute）范式，IR-based方法遵循检索和排名（retrieval and rank）范式。

SP-based方法的基本步骤：

  1. 通过Question Understanding模块对问题进行语法和语义分析并获得编码的问题（Encoded Question）；
  2. 通过Logical Parsing模块将Encoded Question转化为未实例化的逻辑形式，未实例化的逻辑形式是指是问题的句法表示，没有实体和关系的基础；
  3. 通过KB Grounding模块将对结构化知识库进行一些语义对齐，进一步实例化和验证逻辑形式从而可以在知识库执行；
  4. 通过KB Execution模块将解析的逻辑形式在知识库上执行，从而回去预测的答案。

IR-based方法的基本步骤：

  1. 通过Retrieval Source Construction模块从知识库中提取问题特定子图，理想情况下，该图包括所有与问题相关的实体和关系，分别作为节点和边；
  2. 通过Question Representation模块将输入问题进行编码。该模块分析问题的语义并输出推理指令，这些指令通常表示为向量；
  3. 通过Graph Based Reasoning模块通基于向量的计算进行语义匹配，以传播然后沿图中的相邻实体聚合信息。推理状态基于推理指令进行更新（语义匹配）。2、3步骤可重复多次来进行推理。
  4. 通过Answer Ranking模块来根据推理结束时的推理状态对图中的实体进行排序，排名靠前的实体被预测为问题的答案。

SP-based的优缺点（Pros and Cons）：

  • Pros：通过生成表现丰富的逻辑形式来产生更具可解释性的推理过程。
  • Cons：严重依赖逻辑形式和解析算法的设计，这成为性能提升的瓶颈。

IR-based的优缺点（Pros and Cons）：

  • Pros：对图结构进行复杂的推理并执行语义匹配。这种范式自然适合流行的端到端训练，并使IR-based的方法更易于训练。
  • Cons：推理模型的黑盒风格使中间推理的可解释性降低。

挑战和解决方案

![挑战及相应的解决方案](2021-08-20-论文阅读笔记-A-Survey-on-Complex-Knowledge-Base-Question-Answering-Methods-Challenges-and-Solutions/challenge and solution.png)

一、SP-based Method

  • Question Understanding模块

    挑战：理解复杂的语义和语法

    解决方法（方法及论文）：

    1、Adopt structural properties

    • dependency parsing

      • Automated template generation for question answering over knowledge graphs
      • Never-ending learning for open-domain question answering over knowledge bases
      • Knowledge base question answering via encoding of complex query graphs

    • AMR（Abstract Meaning Representation）

      • Question answering over knowledge bases by leveraging semantic parsing and neuro-symbolic reasoning

    2、skeleton-based parsing

    • SPARQA: skeleton-based semantic parsing for complex questions over knowledge bases

    3、structural properties based matching

    • Learning to rank query graphs for complex question answering over knowledge graphs
    • Knowledge-based question answering by tree-to-sequence learning
    • Formal query building with query structure prediction for complex question answering over knowledge base

  • Logical parsing模块

    挑战：解析复杂的查询

    解决方法（方法及论文）：

    Develop expressive targets for parsing

    • template based queries
      • More accurate question answering on freebase
    • query graph
      • Question answering with knowledge base
      • Automated template generation for question answering over knowledge graphs
      • A state-transition framework to answer complex questions over knowledge base

  • KB grounding模块

    挑战：更大的搜索空间

    解决方法（方法及论文）：

    1、Narrow down search space by decompose-execute-join strategy

    • Question answering over knowledge graphs: Question understanding via template decomposition
    • Learning to answer complex questions over knowledge bases with query composition

    2、expand-and-rank strategy

    • UHop:An unrestricted-hop relation extraction framework for knowledge-based question answering
    • Multi-hop knowledge base question answering with an iterative sequence matching model
    • Query graph generation for answering multi-hop complex questions from knowledge bases

  • Training procedure模块

    挑战：在弱监督信号下训练

    解决方法（方法及论文）：

    1、Adopt reward shaping strategy to strengthen training signal

    • Complex program induction for querying knowledge bases in the absence of gold programs
    • Less is more: Data-efficient complex question answering over knowledge bases
    • Hierarchical query graph generation for complex question answering over knowledge graph

    2、conduct pre-training to initialize the model

    • Hierarchical query graph generation for complex question answering over knowledge graph

    3、iterative maximum-likelihood training

    • Neural symbolic machines: Learning semantic parsers on Freebase with weak supervision

二、IR-based Method

  • Retrieval source construction模块

    挑战：在不完全的知识库（子图）下推理

    解决方法（方法及论文）：

    1、Supplement KB with extra corpus

    • Open domain question answering using early fusion of knowledge bases and text
    • Pullnet: Open domain question answering with iterative retrieval on knowledge bases and text

    2、fuse extra textual information into entity representations

    • Improving question answering over incomplete kbs with knowledge-aware reader
    • Open domain question answering based on text enhanced knowledge graph with hyperedge infusion

    3、leverage KB embeddings

    • Improving multi-hop question answering over knowledge graphs using knowledge base embeddings

  • Question representation模块

    挑战：理解复杂的语义

    解决方法（方法及论文）：

    1、Update with reasoned information

    • Key-value memory networks for directly reading documents
    • An interpretable reasoning network for multi-relation question answering
    • Enhancing key-value memory neural networks for knowledge based question answering

    2、dynamic attention over the question

    • Improving multi-hop knowledge base question answering by learning intermediate supervision signals

    3、enrich the question representation with contextual information of graph

    • Open domain question answering using early fusion of knowledge bases and text

  • Graph based reasoning模块

    挑战：推理的不可解释性

    解决方法（方法及论文）：

    1、Provide traceable reasoning path

    • An interpretable reasoning network for multi-relation question answering
    • Enhancing key-value memory neural networks for knowledge based question answering

    2、hyperedge based reasoning

    • Two-phase hypergraph based reasoning with dynamic relations for multi-hop kbqa

  • Training procedure模块

    挑战：弱监督信号下的训练

    解决方法（方法及论文）：

    1、Provide shaped reward as intermediate feedback

    • Stepwise reasoning for multi-relation question answering over knowledge graph with weak supervision

    2、augment intermediate supervision signals with bidirectional search algorithm

    • Improving multi-hop knowledge base question answering by learning intermediate supervision signals

    3、adopt variational algorithm to train entity linking module

    • Variational reasoning for question answering with knowledge graph

结论及未来方向

本文概述了复杂 KBQA 的典型挑战和相应的解决方案，并总结了广泛采用的SP-based的方法以及IR-based的方法。我们相信复杂的KBQA将继续成为一个活跃且有前景的研究领域，具有广泛的应用，例如自然语言理解、组合泛化、多跳推理。

未来方向：

  • Evolutionary KBQA

  • Robust and Interpretable Model

  • More General Knowledge Base
